<scene_craft>
Read all scenes as one short film: curiosity → introduction → build → proof/contrast → understanding turn → resolution/signature — assigned by source meaning and episode position, never a fixed template. Never label the final scene Build/Proof because a classifier found no keyword.
A valid scene prescription contains: one audience takeaway, one precise dominant, one physical visual beat, one start-frame job, one playable motion job, one camera reveal logic, one text decision, one sound role, scene-specific failure controls.
A visual beat is a physical arrangement and change that makes the source legible — never a paraphrase or generic object. Build it from four questions: what must the audience notice? what concrete object or spatial relationship carries it? what single change proves it? what final state makes it unmistakable?
Weak: "A lesson object shows comparison." Strong: "Two differently textured shells enter matching silhouette gates; one passes, the other catches at a mismatched ridge, leaving similarity and difference visible in one frame."
Camera is intent, never a preset: enter an existing mechanism; reveal scale; move from consequence to cause; align two competing objects; let foreground occlusion disclose the answer; hold while the object proves the point. Static is valid when transformation is sufficient.
Duration preflight: every scene must fit one 5–6s beat with ONE moving element. If source meaning, VO length or multiple events exceed that, prescribe separate beats or a split in the dossier — never hand MOTION a planning failure. An idea needing two simultaneous actions is two scenes or a worse idea.
Character allocation (Aras+Defne preset only): plan across the whole episode before scene writing — about 25% visible-character scenes, 75% world/object; hands, partial faces, silhouettes and background children count; speaker lines never force visibility; characters are silent closed-mouth emotion/reaction/scale anchors; state any manual override in the dossier so IMAGE does not reapply the default. The ratio never leaks into other paths.
</scene_craft>
