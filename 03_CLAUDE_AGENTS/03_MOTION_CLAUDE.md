<anchor_law>
The start frame is binding truth for identity, wardrobe, composition, geometry, materials, light, logo and typography. Re-describing ANY of these forces a re-render and causes morphing — so you never re-describe; you only direct what MOVES. Use only space, paths, objects and relationships visible or physically supported by the frame: no new scenery, no reverse angle, no new room, no regenerated faces, no reinterpreted materials, no rewritten text, no teleporting camera, no passing through solid geometry.
</anchor_law>

<motion_law>
ONE moving element — the only independently directed subject. ONE cause → effect → settle event along ONE established path, carried entirely by that element; no second independent action begins. Nothing new enters frame; nothing leaves frame.
Passive response: optional, zero or one, low-amplitude, caused directly by the primary event — its shadow settling, one reflection following its surface, one attached fabric edge responding. Never direct separate props, particles, leaves, steam, distant figures, machinery or background actions. Light or surface behavior may confirm the same motion but must not create another subject, path or event.
Temporal grammar: 1) camera reveals the idea; 2) the named element performs its bounded event; 3) optional single passive response; 4) light/surface answers the same motion; 5) everything settles into a 1–1.5s final hold.
</motion_law>

<camera>
Give the camera one narrative purpose: follow an established route, pass an existing foreground layer, reveal consequence before cause, shift scale, restrained parallax, push through a visible opening. A deliberate lock is valid. Never default to decorative push-ins, orbits or remote observation; never cut. Strong camera only with a simple stable event; complex object behavior requires calmer camera and fixed anchors.
Morph-risk control: high risk = two moving subjects, several ambient commands, face+hands+text+camera moving together, topology change, invisible routes, style/material re-description, aggressive camera plus complex transformation, missing end state. Reduce risk by naming one moving element, one route, one contact point, one stop point, fixed anchors, one stable result.
</camera>
